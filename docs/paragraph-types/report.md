### **1. Text & Narrative Components**
These elements form the core of the site's storytelling and information hierarchy. They are used to create structure, highlight key insights, and manage long-form content.

| Component Label | Description |
| :--- | :--- |
| **Heading** | Standard heading element used to define section titles and hierarchy. |
| **Text** | The primary component for standard body copy and narrative prose. |
| **Accordion** | A vertically collapsing container used to organize and hide dense information. |
| **Article Inline Quote** | A stylized element for highlighting specific quotes or "pull-out" text within an article. |
| **List Item** | A specific entry format used within structured list layouts. |

---

### **2. Media & Data Visualization**
These components are designed to handle visual assets and data-heavy information, ensuring that maps, charts, and high-resolution imagery are displayed correctly.

| Component Label | Description |
| :--- | :--- |
| **Media** | General-purpose wrapper for images, video, or other digital assets. |
| **Figure** | Used for images that require specific captions or instructional framing. |
| **Chart** | A specialized component for rendering interactive or static data-driven charts. |
| **Iframe** | Used to embed external content, such as interactive maps or third-party tools. |
| **Full Width Background Photo** | Displays an image in the background with a foreground content overlay. |

---

### **3. Informational Cards & Callouts**
Card-based components are used to summarize data points or provide quick "at-a-glance" information to users scanning a page.

| Component Label | Description |
| :--- | :--- |
| **Stat Card** | A high-visibility card used to feature individual key metrics or statistics. |
| **Facts** | A container designed for a collection of quick, digestible data points. |
| **Findings Card** | Specifically designed for featuring research results or study highlights. |
| **Goals Card** | A stylized card used to showcase specific strategic objectives or targets. |
| **Block Card** | A standard container for grouping related content and links. |
| **Large Card With Photo** | A prominent feature card typically used for high-level navigation. |
| **About DVRPC Callout** | A standardized block providing organizational context and boilerplate. |
| **Datacenter Callout** | A specific callout directing users toward data-related services or tools. |

---

### **4. Interaction & Navigation**
These components drive user engagement by providing clear calls to action, resource links, and downloadable documentation.

| Component Label | Description |
| :--- | :--- |
| **Button** | A standalone action link with various styling options. |
| **Button Group** | Contains one or more buttons with specific alignment and layout options. |
| **Call-To-Action** | A high-priority directive designed to guide user behavior toward a specific goal. |
| **Document with Preview** | Displays a file for download alongside a visual thumbnail or preview. |
| **Related Resources** | Specific cards used for supplementary links on Topic landing pages. |
| **Small Callout With Icon** | A compact callout box featuring a descriptive icon. |
| **Small Callout With Image** | A compact callout box featuring a small illustrative image. |

---

### **5. Structural & Specialized Blocks**
These components handle the underlying layout of the page or pull in dynamic data from other parts of the website database.

| Component Label | Description |
| :--- | :--- |
| **Section** | The primary layout wrapper, supporting 1, 2, 3, or 4-column configurations. |
| **Inner Section** | An additional layout section used for nesting columns within a parent Section. |
| **From library** | Allows for the insertion of pre-configured, reusable content from a central library. |
| **News Block** | A dynamic list that automatically displays the most recent news items. |
| **Events Block** | A dynamic list that displays upcoming organizational events. |
| **Person** | A formatted display for staff profiles, including contact info and titles. |
| **View** | A component that embeds a dynamic list of content based on specific database queries. |
